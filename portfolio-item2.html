Ticket Calculator in Java

This application adds a calculator fuctionality to the app

This was my second application I made in android studio. It allows the user to choose a concert they would like to attend and then choose how many tickets they need.

This application was more advanced than the last as it added a calculator to the app. I had to use my knowledge of java to create a script that adds up the needed amount of tickets and multiplies that by the price of the tickets. I was able to add some more flare to the application as I learned more throughout my class.
